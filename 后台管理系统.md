# 后台管理系统

## 技术
  vue2 + vue-router + Vuex + axios + elementui + echarts

## 项目准备
iconfont

## 项目搭建
1.vue create vue2_admin
2.vue-router vuex
3.axios
4.elementui ---(按需见官网)
创建插件可以按需循环注册element.js,在main.js调用这个方法即可


## 项目初始化
1.删除无用的组件
2.css初始化
reset.css(淘宝方案)，在main.js导入
3.引入iconfont图标
先下载，再在main.js引入
 

## 后台服务 （serve文件）
1.npm i express -S
2.npm i mysql -S
3.jwt 生成token jsonwebtoken
解析token： 安装jwt-decode
4.mockjs ---模拟数据
